Mẫu quy tắc cho tính năng Chat gia đình (Firestore + Storage).
Dán vào Firebase Console → Firestore / Storage → Rules, sau khi chỉnh cho khớp project.

--- Firestore (thêm vào match families/{familyId}) ---

  match /chatMessages/{messageId} {
    function isSignedIn() {
      return request.auth != null;
    }
    function memberFamilyId() {
      return get(/databases/$(database)/documents/users/$(request.auth.uid)).data.familyId;
    }
    function isFamilyMember() {
      return isSignedIn() && memberFamilyId() == familyId;
    }
    allow read: if isFamilyMember();
    allow create: if isFamilyMember()
      && request.resource.data.userId == request.auth.uid
      && request.resource.data.keys().hasAll(["userId", "authorName", "text", "attachment", "createdAt"]);
    allow delete: if isFamilyMember() && resource.data.userId == request.auth.uid;
    allow update: if false;
  }

--- Storage ---

rules_version = '2';
service firebase.storage {
  match /b/{bucket}/o {
    match /families/{familyId}/chat/{messageId}/{fileName} {
      allow read: if request.auth != null
        && firestore.get(/databases/(default)/documents/users/$(request.auth.uid)).data.familyId == familyId;
      allow write: if request.auth != null
        && firestore.get(/databases/(default)/documents/users/$(request.auth.uid)).data.familyId == familyId
        && request.resource.size < 13 * 1024 * 1024;
    }
  }
}

Lưu ý:
- Đổi (default) nếu database ID khác.
- create rule có thể nới lỏng/bỏ hasAll nếu schema đổi.
- Chưa có rule update tin nhắn (ứng dụng chỉ tạo/xóa).
